Usando marcação de texto aqui com <mark> e </mark>

testando se esta palavra fica pequena usando <small> e </small>

para excluir um texto use a tag <del> e </del>

para incluir um texto use a tag <ins> e </ins>

para sobrescrever um texto use a tag <sup> e </sup>, exemplo: X2

para subscrever um texto use a tag <sub> e </sub>, exemplo: H2O

Outras formatações

para adicionar uma pre formatação de texto use a tag <pre> e </pre>
exemplo:

  testando aqui
  123
  testando

para adicionar espacamento entre os textos use a tag <code> e </code>
exemplo:

testando os espacamentos 123

para citações simples use a tag <q> e </q>
exemplo:

citando aqui

para citações compostas (com destaque) use a tag <blockquote> e </blockquote>
